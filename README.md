# Todo app

Aplicacion TODO's en linea JAMSTACK



## Tech Stack

**Client:** Nextjs, TailwindCSS, Auth0, Airtable

## Features

-   Registro/Iniciar sesion

-   CRUD app

## Ejecutar localmente

Clonar proyecto

```bash
  git clone https://github.com/kioshiokamoto/nextjs-jamstack-todo
```

Ir a ruta de proyecto

```bash
  cd nextjs-jamstack-todo
```

Instalar dependencias

```bash
  npm install
```

Ingresar variables de entorno en .env

```bash
  AIRTABLE_API_KEY=
  AIRTABLE_BASE_ID=
  AIRTABLE_TABLE_NAME=
  AUTH0_DOMAIN=
  AUTH0_SECRET=
  AUTH0_CLIENT_ID=
            =
```

Iniciar servidor

```bash
  npm run start
```
